# 🔐 Página de Login e Cadastro

Este projeto consiste em uma **página de Login e Cadastro**, desenvolvida com **HTML5, CSS3 e JavaScript**, focada em usabilidade, layout moderno e interatividade no front-end.

## 📌 Objetivo do Projeto
O objetivo deste projeto é praticar e demonstrar habilidades em desenvolvimento front-end, incluindo:
- Estruturação semântica com HTML5  
- Estilização moderna e responsiva com CSS3  
- Manipulação do DOM e interatividade com JavaScript  
- Criação de formulários funcionais de login e cadastro  

## 🛠️ Tecnologias Utilizadas
- HTML5  
- CSS3  
- JavaScript  

## ✨ Funcionalidades
- Formulário de **Login**
- Formulário de **Cadastro**
- Alternância entre Login e Cadastro via JavaScript  
- Validação básica de campos  
- Layout moderno e intuitivo  
- Interface responsiva para diferentes dispositivos  

## 📱 Responsividade
O layout foi desenvolvido para se adaptar a diferentes tamanhos de tela, como:
- Desktop  
- Tablets  
- Smartphones
